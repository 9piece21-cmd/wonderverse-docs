## 1. 欢迎来到 Wonderverse

Wonderverse 是一个面向漫剧、游戏与电商等多种场景的端到端 AIGC 创作平台。
你只需要一个想法，就能在一处完成从生成、编辑到交付的全部环节。

> *一键全流程生成，轻松将创意转化为完整故事、成片视频或营销活动方案。*

### Wonderverse 是什么

传统内容生产链路里，团队要在十几款工具之间切换——写脚本一处、生图一处、配音一处、剪辑一处、做多语言版本又得绕一圈。素材分散、版本失控、跨工具协作的成本，比创作本身还高。

Wonderverse 把这条链路**收进一个工作室**。

- **底层** 是 7 个原子级 AI 工具：图片生成、视频生成、超分增强、字幕生成 / 译制、AI 译制配音、横转竖、智能擦除。
- **中层** 是 **无限画布**——节点式工作流，让你把上述工具自由编排、保存、复用。
- **上层** 是 4 套**开箱即用的场景方案**：AI 漫剧、游戏高光集锦、电商内容创作、营销视频。

[配图：双层架构示意图——下层 AI 工具，中层 无限画布，上层 场景方案]

### 为什么选 Wonderverse

#### 🎯 端到端，而非碎片拼接
从脚本到成片在同一画布上完成，省掉 80% 的工具切换。

#### 🌏 中日英三语原生
界面、配音、字幕、本地化全链路支持中 / 日 / 英，无需借助第三方翻译。

#### 🧩 场景方案，开箱即用
不必从零学习节点编排——直接打开 AI 漫剧、电商创作等模板，5 分钟出第一版成片。

#### 🔁 资产与工作流可沉淀
团队的风格、角色、节点流可复用，越用越快、越用越像"你们家"的内容。

#### 🤝 团队协作友好
画布可分享、可二创，权限分明，适配 5 - 50 人的中型内容团队。

## 2. 适合谁用

Wonderverse 聚焦在三类规模化生产内容的 B 端团队，**也欢迎 P 端和 C 端创作者体验**。
请对照下方场景，找到你的最佳入口。

### 🎬 AI 漫剧 / 短剧公司

**典型痛点**：剧本到成片链路长，单集制作周期 3-7 天；多集拍摄角色 / 画风容易漂移；海外发行需要重做配音字幕。

**推荐入口**：
- 🎯 [AI 漫剧生成 Agent](#15--ai-漫剧生成-agent) — 从剧本到成片端到端流水线
- 🎙️ [AI 译制配音](#11-ai-译制配音) — 中日英多语版本一键产出
- 📐 [横转竖](#12-横转竖) — 横屏母带智能改竖

**典型工作流**：粘贴剧本 → 资产库锁角色 → AI 分镜 → 批量生成 → 多语言导出

---

### 🎮 游戏公司

**典型痛点**：赛事 / 直播素材数小时起步，人工剪辑赶不上节奏；多平台分发需要反复重剪；新版本宣发素材产能跟不上版本节奏。

**推荐入口**：
- 🎯 [游戏 / 比赛高光集锦](#16--游戏--比赛高光集锦) — 长视频自动剪辑成短片
- 🎬 [视频生成](#7-video-generation--生视频) — 角色 / 场景概念片
- 📝 [字幕生成 / 译制](#10-字幕生成--译制) — 选手语音转字幕，海外频道直发

**典型工作流**：导入赛事素材 → AI 识别高光时刻 → 自动剪辑 + 字幕 → 多平台规格批量导出

---

### 🛒 电商公司（含跨境）

**典型痛点**：商品图拍摄成本高、迭代慢；模特、场景、风格难以快速 A/B；海外站点需要本地化模特 / 文案 / 配音。

**推荐入口**：
- 🎯 [电商内容创作](#17--电商内容创作) — 商品图、模特、场景一键生成
- 🎨 [生图](#7-生图) — 营销主图与 KV
- 📢 [营销视频](#18--营销视频) — 短视频广告批量产出

**典型工作流**：上传商品图 → 选品类风格 → 批量生成多角度 / 多场景 / 多语言版本 → 直接对接投放平台

## 3. 5 分钟快速上手

第一次打开 Wonderverse？根据你今天想做的事，挑一条路开始——

> 🚀 **想快速生成一张图或一段视频？** 走「创作」工作台，3 步出片
> 🎨 **想搭一条可复用的工作流？** 进「无限画布」，5 步玩转节点连线

---

### 路径一 · 用「创作」3 步出片

适合：第一次用、想最快看到效果。

#### 第 1 步 · 打开「创作」
登录后从左侧菜单点击 **「创作」**。

#### 第 2 步 · 选生成类型与模型
顶部 Tab 切到 **生成**，再选 **图像** 或 **视频**。
在 Model 区下拉选择你想用的模型（推荐 Gemini 3.0 / Kling 3.0）。

#### 第 3 步 · 写 prompt + 运行
- 在描述框里写你的 prompt
- 上传参考图（可选，最多支持 14 张，具体数量随所选模型而异）
- 选画幅、画质、批次
- 点击 **运行**，几十秒后中央预览区出现成品

> 想做编辑？切到 **编辑** Tab，调用超分、字幕、配音、横转竖、智能擦除。

[配图：创作工作台主界面]

---

### 路径二 · 用「无限画布」搭一条工作流

适合：想理解节点流，做更复杂、可复用的生产线。

#### 第 1 步 · 创建一张画布
登录后，从左侧菜单点击 **「无限画布」** → **「新建空白画布」**。
进入后在顶部点击项目名重命名。

> 💡 **不想从空白开始？** 同一入口里有内置的推荐案例——漫剧、电商、社媒等简单工作流模板都可以一键打开，跟着改最快。

#### 第 2 步 · 加入第一个节点（生图）
1. 左上角点击 **「＋ 添加节点」** 按钮，弹出节点选择面板
2. 选择 **「生图」**，节点出现在画布上
3. 在节点的输入框里写 prompt，比如：
   > *"一只英短猫蹲在窗台，黄昏的光线斜射进来，电影感色调，35mm 镜头"*
4. 在 **输入框左下角** 选择你想要的生图模型（主流模型都已接入）
5. 点击节点上的 **▶ 运行**

约 10-30 秒后，节点中央会出现生成的图片。

#### 第 3 步 · 加入第二个节点（生视频）
再次点击左上角 **「＋ 添加节点」**，选择 **「生视频」**，放在生图节点的右边。

#### 第 4 步 · 连接节点 · 让图变成视频的首帧
这一步是画布的核心操作——**拖连线**。

1. 鼠标按住生图节点右边的 **输出端口●**
2. 拖动到生视频节点左边的 **输入端口●**，松开
3. 画布上出现一条连线，表示**上游的图被作为下游的首帧图传入**

> 💡 这就是"无限画布"的精髓——任意两个节点之间只要类型匹配，就能用一根线打通。

接着在生视频节点里：
- 输入框写一句关于**运动**的描述：*"猫缓慢转头，看向镜头，发丝随风轻拂"*
- 输入框左下角选择视频生成模型
- 参数面板调整：**画幅 9:16 / 时长 5s**
- 点 **▶ 运行**

[配图：连线动作演示 + 节点参数面板]

#### 第 5 步 · 预览与导出
1. 等待约 60-120 秒，生视频节点出现成片
2. 点击节点中央预览全屏播放
3. 满意之后，点击画布**右上角**的 **「导出案例」** 按钮，一次性导出当前项目的所有产物（图 + 视频）

🎉 你已经完成了第一条画布工作流——从一张 AI 图，到一段以它为首帧的 AI 视频。

- 把这条工作流**保存为模板**，下次直接复用（见 [创建工作流](#21-创建工作流)）

## 4. 团队与权限

Wonderverse 为 5 - 50 人的中型内容团队设计了一套**轻量但能落地**的协作机制。
你可以快速创建团队、邀请成员、分配权限、共享资产，让创作流程透明可控。

### 创建团队

1. 个人控制台右上角 → **创建团队**
2. 填写团队名 + Logo（可选）
3. 选择团队套餐〔参考 [计费与发票](#28-计费与发票)〕

### 邀请成员

进入 **团队设置 → 成员**：

- 通过邮箱邀请（最多批量 50 个）
- 生成邀请链接（可设置有效期 / 上限人数）
- 通过 OAuth 接入企业 SSO〔企业版〕

### 角色与权限

| 角色 | 项目权限 | 资产权限 | 计费权限 |
|---|---|---|---|
| **🛡 管理员 Admin** | 创建 / 删除 / 全部访问 | 全部读写 | ✅ |
| **🎨 创作者 Editor** | 创建 / 编辑自己 + 团队共享 | 读 + 写自己创建的 | ❌ |
| **👀 观察者 Viewer** | 只读访问已分享项目 | 只读 | ❌ |
| **🤝 客座 Guest** | 单项目临时只读 | ❌ | ❌ |

### 项目共享

每个项目有三档可见性：

- 🔒 **私有** —— 仅创建者可见
- 👥 **团队共享** —— 团队成员按角色权限访问
- 🌐 **公开链接** —— 任意持有链接者可只读访问（适合外部审稿）

### 资产中心

团队的角色库、声音库、风格库、模板库统一沉淀在**资产中心**：

- 任何成员保存的资产，可一键升级为**团队资产**供全员调用
- 资产支持版本管理与回滚
- 管理员可在 **资产中心 → 治理** 里设置审核流（企业版）

### 数据安全〔企业版〕

- 私有部署 / VPC 网络隔离〔待核实〕
- 国密合规 / GDPR 合规支持〔待核实〕
- 操作审计日志，全行为可追溯〔待核实〕

## 7. 生图

通过文字描述生成图片。Wonderverse 的生图模型在**电商商品**、**漫剧风格**、**营销 KV** 三类场景做了专项优化。

### 能力概览

- ✅ 生图（纯 prompt）
- ✅ 图生图（参考图 + prompt）
- ✅ 多图融合（最多支持 14 张参考图，具体数量随所选模型而异）
- ✅ 多种内置风格（日漫、3D 写实、水墨、赛博朋克、Q 版等）

### 输入

| 输入项 | 说明 |
|---|---|
| **Prompt（必填）** | 文字描述，最长 1000 字 |
| **参考图（可选）** | 最多支持 14 张，具体数量随所选模型而异 |
| **负向 Prompt（可选）** | 描述你**不想要**的元素 |

### 参数

| 参数 | 选项 | 默认 |
|---|---|---|
| **风格预设** | 日漫 / 3D 写实 / 水墨 / 赛博朋克 / Q 版 / 写实摄影 / 自定义 | 写实摄影 |
| **画幅** | 1:1 · 4:3 · 3:4 · 16:9 · 9:16 · 自定义尺寸 | 1:1 |
| **分辨率** | 标清 / 高清 / 超高清 | 高清 |
| **种子 Seed** | 任意整数；同 prompt + 同 seed 出同一张图 | 随机 |
| **执行次数** | 1× / 2× / 4× | 1× |

### prompt 速成

一个高质量 prompt 通常包含 5 个要素：

```
[主体] + [动作 / 状态] + [环境 / 时间] + [视觉风格] + [镜头 / 光线]
```

**示例**：
> 一位身着和服的少女（主体）轻倚樱花树（动作）+ 京都春日清晨（环境）+ 浮世绘风格（风格）+ 长焦虚化背景（镜头）

更多技巧请见 [提示词技巧](#24-提示词技巧)。

### 常见问题

**Q：参考图最多几张？**
A：最多支持 14 张，具体数量随所选模型而异。

**Q：能锁定生成同一个角色吗？**
A：可以，通过 [个人资产](#22-个人资产) 把角色保存为资产卡，每次调用即生成同一形象。

**Q：商用版权如何处理？**
A：付费用户生成的图片享有商用版权〔细则待核实〕。

## 8. 生视频

通过文字描述（可选首帧 / 首尾帧）生成视频。
是 Wonderverse 最核心的视频生产引擎，也是漫剧 / 高光 / 营销视频场景的发动机。

### 能力概览

- ✅ 纯生视频
- ✅ 图生视频（单张作首帧）
- ✅ 首尾帧生视频（两张图分别作首尾，AI 补中间过渡）
- ✅ 视频延长（基于已有视频末尾继续生成）
- ✅ 音画同出（部分模型支持，如 Kling 3.0 Omni，直接生成自带音频的视频）

### 输入

| 输入项 | 说明 |
|---|---|
| **Prompt（必填）** | 描述镜头内容，最长 1500 字 |
| **参考图（可选）** | 根据模型而定，最多 4 张 |
| **首帧图（可选）** | 锁定第一帧画面 |
| **尾帧图（可选）** | 与首帧搭配，AI 补中间运动 |
| **负向 Prompt（可选）** | 描述你**不想要**的元素 |

### 参数

| 参数 | 选项 | 默认 |
|---|---|---|
| **时长** | 4s · 5s · 6s · 8s · 10s · 12s（不同模型支持的可选项不同） | 5s |
| **画幅** | 16:9 · 9:16 · 1:1（不同模型支持的可选项不同） | 16:9 |
| **分辨率** | 720p · 1080p | 720p |
| **风格预设** | 与生图一致 | 写实 |

### prompt 注意

视频 prompt 与图像 prompt 的区别：**必须描述运动**。

```
✅ 推荐：镜头缓慢推进，少女转头微笑，发丝随风轻拂
❌ 不推荐：一个微笑的少女
```

更多视频 prompt 技巧请见 [提示词技巧 · 视频篇](#24-提示词技巧)。

> 💡 想要更高画质？跑完视频后接「编辑」栏目里的「超分增强」，可获得 1080P / 2K / 4K 输出。

## 9. 超分增强

把模糊、低分辨率、噪点重的视频或图片提升画质。
适合**老素材修复**、**用户上传低质素材改造**、**生成结果二次精修**三类场景。

### 能力概览

- ✅ 视频增强（4 档输出：720P / 1080P / 2K / 4K）
- ✅ 图片增强（3 种增强模式：标准 / 人像 / 暗光）
- ✅ 内容类型自动适配（漫剧 / 真人 / 通用）
- ✅ 小脸优化（视频专属，人脸偏小时更好还原五官）

### 输入

| 类型 | 支持格式 |
|---|---|
| 🎬 **视频** | MP4 / MOV / WebM，建议 ≤ 5 分钟 |
| 🖼️ **图片** | JPG / PNG / WebP |

### 视频参数

#### 内容类型（按画面性质选）

| 选项 | 描述 |
|---|---|
| 🎭 **漫剧 / 动漫** `推荐` | 动画风格、漫剧场景专项优化 |
| 👤 **真人** | 真人出镜，肤色 / 细节优化 |
| 🎨 **通用** | 不确定时选这个，适合各类画面 |

#### 增强档位（决定输出分辨率与精修强度）

| 档位 | 输出 |
|---|---|
| ⚡ **快速修复** | 720P |
| 🎯 **标准修复** | 1080P |
| ✨ **极致增强** | 2K |
| 🎬 **影院级** | 4K |

#### 小脸优化

开关。人脸偏小时更好还原五官。

### 图片参数

#### 增强模式

| 选项 | 描述 |
|---|---|
| 📐 **标准** `默认` | 适合大部分图片 |
| 👤 **人像** | 真人 / 自拍，开启人脸增强 |
| 🌃 **暗光** | 夜景 / 室内偏暗，提升亮度 |

### 典型用法

| 场景 | 推荐设置 |
|---|---|
| 漫剧老素材修复 | 内容类型「漫剧 / 动漫」+ 极致增强 2K（或影院级 4K） |
| 真人短剧 / 直播录像 | 内容类型「真人」+ 标准修复 1080P + 开启小脸优化 |
| 不确定 / 混合画面 | 内容类型「通用」+ 标准修复 1080P |
| 老照片 / 模糊图片 | 增强模式「标准」（图片） |
| 自拍精修 | 增强模式「人像」（图片） |
| 夜景暗调照片 | 增强模式「暗光」（图片） |

## 11. AI 译制配音

把视频里说一种语言的人，原音原色地说另一种语言。
30 种源语言互译，音色克隆 + 字幕同步压制。

### 能力概览

- ✅ 自动识别视频中的语音
- ✅ 翻译到目标语言（30 种）
- ✅ 音色克隆：用原视频里的声音说目标语言
- ✅ 翻译后字幕可选烧录
- ✅ 原视频自带字幕自动擦除
- 🚧 配音库选择、语速 / 情感 / 音调精细控制（即将上线）

### 输入

| 类型 | 支持格式 | 限制 |
|---|---|---|
| 视频 | MP4 / MOV / WebM | 建议 ≤ 5 分钟 |

### 参数

#### 视频是否自带字幕？

| 选项 | 处理逻辑 |
|---|---|
| 🎬 **有，画面上可以看到字幕** `默认` | 自动识别字幕位置 → 擦除原字幕 → 压新字幕 |
| 🎙️ **没有，只有人声** | 通过语音识别自动生成字幕 |

#### 源语言（30 种）

简体中文 · 繁體中文 · 英语 · 日语 · 韩语 · 法语 · 德语 · 西班牙语 · 葡萄牙语 · 意大利语 · 俄语 · 阿拉伯语 · 泰语 · 越南语 · 印度尼西亚语 · 马来语 · 印地语 · 土耳其语 · 波兰语 · 荷兰语 · 瑞典语 · 丹麦语 · 芬兰语 · 挪威语 · 捷克语 · 希腊语 · 匈牙利语 · 罗马尼亚语 · 乌克兰语 · 希伯来语

#### 目标语言（同样 30 种）

#### 压制翻译后的字幕

开关。开启 = 把翻译后的字幕烧录到画面里。
若原视频自带字幕，会先自动擦除再压新字幕。

### 🚧 配音音色（即将上线）

| 模式 | 说明 |
|---|---|
| 🎤 **音色克隆** | 用原视频说话人的声音说目标语言（当前默认行为） |
| 🎙️ **配音库选择** | 从内置音色库选一个适配的声音 |

### 🚧 精细参数（即将上线）

| 参数 | 选项 |
|---|---|
| 语速 | 0.5× ~ 2× |
| 情感 | 自然 / 喜悦 / 悲伤 / 愤怒 / 惊讶 / 严肃 |
| 音调 | 低 / 中 / 高 |
| 多角色分配 | 自动 / 手动指定每个角色对应的音色 |

### 🚧 自定义声音克隆（即将上线）

当前版本会**自动**从原视频中克隆说话人的声音用作配音。
未来版本将支持手动上传自定义克隆样本，建议：

- 样本时长 **30 秒**（≥ 20 秒可用，越长越像）
- **干净人声**：无背景音乐、无回声、无杂音
- **中性内容**：避免情绪极端，AI 提取的是音色不是情绪
- 单一说话人

### 典型用法

| 场景 | 推荐设置 |
|---|---|

| 跨境电商带货 | 源：简体中文 / 目标：英语 / 字幕压制：开 |
| 海外 KOL 内容本地化 | 源：英语 / 目标：简体中文 / 字幕压制：开 |

### 第 1 步 · 输入剧本

进入 **左侧菜单「AI 短剧」**，主界面是一个对话框。你有两种方式提供剧本：

#### 选项 A · 粘贴现成剧本
直接粘贴 Word、TXT 或纯文本剧本。支持长文本（**单次最长 10 万字**），Agent 会自动按集数切分。

#### 选项 B · 让 AI 帮你写
输入一句故事 logline，例如：
> *"东京涉谷，一个失忆的退役职业杀手，在便利店遇见了自己的女儿。"*

Agent 会按你的设定生成完整剧本。

#### 配置项
| 配置 | 说明 |
|---|---|
| 🎨 **风格** | 日漫、3D 写实、水墨、赛博朋克、Q 版等多种内置风格 |
| 📺 **集数** | 1 - 100 集 |
| ⏱️ **单集时长** | 60s / 90s / 120s / 180s |
| 🎞️ **画幅** | 9:16 竖屏 · 16:9 横屏 · 1:1 方形（可多选） |
| 🌏 **语言** | 中 / 日 / 英 |

[配图：剧本输入对话框 + 配置面板]

### 第 2 步 · 构建资产库

剧本提交后，Agent 自动拆解出三类资产：

#### 🧑 角色
每个出场人物自动生成**形象卡 + 性格卡 + 声音卡**。你可以：
- 替换角色形象（上传参考图 / 重新生成 / 从资产库挑选）
- **克隆真人声音**（上传 30 秒清晰人声样本即可）
- 从内置音色库选择（含中日英多种音色）

#### 🏞️ 场景
按拍摄地点自动归类，生成主视觉。可调整：
- 时间（清晨 / 黄昏 / 深夜）
- 天气（晴 / 雨 / 雪 / 雾）
- 镜头氛围（暖 / 冷 / 高对比）

#### 🎭 道具
关键道具（武器、信物、关键物件）单独管理，确保多镜头一致。

> 💡 **资产可沉淀** — 本集做好的角色、声音、场景，可一键保存到团队资产库，下一部直接复用。

### 第 3 步 · AI 分镜

Agent 把剧本逐段切分为镜头，输出一张**分镜表**，每个镜头包含：

| 字段 | 示例 |
|---|---|
| 镜号 | 01-007 |
| 景别 | 中景 / 特写 / 大全景 |
| 角色 | 主角 A + 路人 B |
| 场景 | 涉谷便利店 · 雨夜 |
| 动作 | 主角 A 缓慢转身，目光落在收银台 |
| 台词 | "原来…是你。" |
| 时长 | 3.2 秒 |

每一行都可以**单独编辑**——改文案、换镜头、调时长、加注解。

### 第 4 步 · 生成视频

确认分镜后，点击 **生成**：

- **单镜生成**：逐个镜头预览，不满意可重抽卡
- **批量生成**：整集一键全跑，Agent 自动排队
- **整剧生成**：100 集级别可后台批量产出（**夜间批跑友好**）

> 视频生成支持 720p / 1080p（想要更高画质，跑完后接「编辑」栏目里的「超分增强」）；每镜头可设置 1× / 2× / 4× 抽卡次数。

### 第 5 步 · 剪辑与导出

视频跑完后进入剪辑台：

- **基础剪辑**：裁剪、拼接、转场、字幕、BGM
- **多画幅导出**：横竖方一次性导出，自动重构图（基于横转竖）
- **多语言版本**：勾选目标语种，自动调用 AI 译制配音 + 字幕生成 / 译制，一键产出本地化版本
- **导出格式**：MP4 (H.264 / H.265)、MOV、最高 1080p

> 🚧 **剪辑功能持续迭代中** — 高级时间轴、多轨混音、AI 自动剪辑等功能正在路上。

### 完整示例

> **场景**：日本电商品牌"花鸟集"想做一部 12 集竖屏漫剧，宣传新品香水线，目标投放 TikTok 日区。

| 阶段 | 操作 | 耗时 |
|---|---|---|
| 输入 | 粘贴 8000 字剧本，选「日漫风格 / 9:16 / 90s / 日语」 | 2 分钟 |
| 资产库 | 接受默认角色，替换 2 个场景，克隆声优样本 | 8 分钟 |
| 分镜 | 调整 6 个镜头的景别与台词 | 10 分钟 |
| 生成 | 12 集后台批量生成 | 一夜 |
| 剪辑 | 添加品牌片头 + 字幕，导出竖屏 1080p | 15 分钟 |
| **总计** | **从剧本到 12 集成片** | **约 4 小时人工 + 一夜批跑** |

### 常见问题

**Q：剧本太长会被截断吗？**
A：单次最长 10 万字，超过会自动按章节切分多个项目管理。

**Q：能保证整剧角色一致吗？**
A：通过资产库锁定角色形象 + 声音 ID，整部剧调用同一资产，确保跨集一致。

**Q：可以中途回到上一步修改吗？**
A：可以。每一步都是独立节点，修改后**只重跑下游**，不会浪费已生成的素材。

**Q：声音克隆需要多长样本？**
A：30 秒清晰中性人声即可（无背景音乐 / 无回声）。

### 第 2 步 · 生成脚本

Agent 输出 3-5 个版本的脚本，每版本含：

- 钩子（前 3 秒）
- 卖点演绎（5-15 秒）
- CTA 行动召唤（最后 3 秒）

支持手动修改 + 锁定特定段落不变。

### 第 3 步 · 自动分镜 + 视频生成

确认脚本后：

- 自动生成分镜
- 调用 [生视频](#8-生视频) 批量出片
- BGM 库自动匹配（按风格）

### 第 4 步 · 配音 + 字幕

- 内置营销级音色库（亲切 / 燃向 / 高级感）

- 字幕样式可绑定品牌字体 / 色卡

### 第 5 步 · 多平台批导

| 平台 | 规格 | 时长 |

| 抖音 / TikTok | 9:16 · 1080p | 9s / 15s / 30s |

| YouTube Pre-roll | 16:9 · 1080p | 6s / 15s / 30s |

## 19. 认识画布

**无限画布** 是 Wonderverse 的创作主战场。
你之前所有的"剧本 → 素材 → 剪辑"都发生在一张张孤立的工具页里；
在这里，它们汇聚到**同一张可以无限延展的画布**上，由你串联、由 AI 接力。

### 它和传统创作工具有什么不同

| 维度 | 传统工具 | 无限画布 |
|---|---|---|
| **基本单位** | 文件、图层、时间轴 | **节点**（一个 AI 能力 = 一个节点） |
| **创作思路** | 线性流水：A 做完交给 B | **网状协作**：多分支并行，结果即时回传 |
| **复用方式** | 复制粘贴模板 | **保存工作流**，下次一键复跑 |
| **协作方式** | 文件来回发送 | 画布共享，谁都能看到全流程 |
| **试错成本** | 改一处，全链路重做 | 改一处，只重跑下游节点 |

### 画布的三个核心概念

#### 🧩 节点 Node
画布上的最小操作单元。每个节点封装一项 AI 能力——比如「生图」「智能擦除」「AI 译制配音」。
节点有输入、有输出，像积木一样可以串接。

#### 🔗 工作流 Workflow
把若干节点用连线组合起来，就是一条**工作流**。
工作流可以保存、命名、复用、分享，是你团队真正的生产力资产。

#### 📦 资产 Asset
画布上沉淀下来的成果——素材、角色、风格、声音、模板——
都归入**资产中心**。资产可被任意节点调用，让团队的风格在所有项目里延续。

### 节点的种类

Wonderverse 内置 7 大类节点，覆盖所有底层 AI 能力：

| 节点类型 | 输入 | 输出 | 用途 |
|---|---|---|---|
| 🎨 **生图** | 文本 prompt | 图片 | 创意素材、分镜稿、海报 |
| 🎬 **生视频** | 文本 prompt（可选首帧图） | 视频 | 短视频片段、镜头素材 |
| 🔍 **超分增强** | 图片 / 视频 | 高清图片 / 视频 | 2× 放大 · 去噪 · 调色 |
| 📝 **字幕生成 / 译制** | 视频 | 多语字幕文件 / 烧录后视频 | 多语言本地化 |
| 🎙️ **AI 译制配音** | 文本 + 音色 | 音频 | 多语言旁白与对白 |
| 📐 **横转竖** | 横屏视频 | 竖屏视频 | 平台适配 |
| 🩹 **智能擦除** | 视频 / 图片 + 选区 | 干净版本 | 去除水印 / 路人 / 物件 |

> 💡 后续会接入更多节点（音乐生成、长视频拼接等），所有节点共享同一套交互逻辑。

### 节点的输入：三种来源

#### 1. 直接输入
手动写指令或上传文件。最常见的用法。

#### 2. 引用上游节点
**用连线连接上游节点的输出端口**，下游就会自动拿到上游的结果。
> 例：把"生图"节点连到"超分增强"节点，前者出图，后者自动放大。

#### 3. 调用资产库
从右侧资产面板拖入角色、声音、风格，节点会把它当作参考输入。
> 例：拖入"主角 A 形象卡"到生图节点 → 生成图里都是同一角色形象。

### 三个让节点更好用的技巧

#### ⚙ 进入参数面板调细节
默认参数适合 80% 场景，但你可以在 ⚙ 面板里调整：
模型版本、风格强度、画幅、种子值、引导系数等。

#### 🔁 用执行次数"抽卡"
同一指令试 4 次，挑最满意的一张。
比反复手动重跑省时间，也省 token。

### 进阶 · 分支与合流

> 例：3 个分镜视频 → 视频拼接节点 → 完整短片〔节点能力待核实〕

### 保存与复用

1. 顶栏点击 **保存为工作流**
2. 设置名称、封面、说明
3. 选择可见性：私有 / 团队 / 公开

下次调用：

- **从模板新建项目** → 直接基于此工作流开始
- **拖入到新画布** → 在已有项目里嵌入这条流
- **API 调用**〔企业版〕 → 把工作流当作服务接入业务系统

### 工作流模板库

- 🎬 漫剧分镜流（含日漫 / 写实 / 水墨变体）

- 🛒 电商主图批量流

模板可一键 fork 到自己的画布修改。

## 23. 分享与协作

Wonderverse 的画布天生为**协作而生**——
不再有"邮件来回发文件"的低效，每个人都能在同一张画布上看到、改动、复用所有流程。

### 三种分享方式

#### 1. 团队内共享
项目设置 → 共享给团队 → 设置每个成员的访问权限：

- **可编辑**：完整修改
- **可评论**：留言但不改动
- **只读**：观摩学习

#### 2. 公开链接分享
生成可访问链接：

| 选项 | 说明 |
|---|---|
| **可见性** | 仅查看 / 可复制（复制到对方账号继续编辑） |
| **有效期** | 24 小时 / 7 天 / 永久 |
| **密码保护** | 可选 |
| **水印** | 选项加上"by 团队名"水印 |

#### 3. 二创（Fork）
对方打开你的公开画布后，可点击 **复制到我的工作区** —— 在自己账号里得到一份独立副本继续改造。

### 实时协作

多人同时打开同一张画布时：

- 各成员的光标 / 当前选中节点实时显示
- 谁改了哪个节点会有头像标记
- 评论可锚定到具体节点，沟通有上下文

### 评论与审稿

#### 节点评论
右键节点 → 添加评论。评论可 @ 指定成员，会发邮件 / 站内通知。

#### 项目审稿
把项目共享给"客座（Guest）"角色，对方只能查看 + 评论，不能改动——
适合给甲方 / 老板看初版的工作流。

### 协作中的版本管理

- 自动保存：每 30 秒自动留存
- 历史版本：右上角 ⏰ 查看任意时刻的画布快照
- 一键回滚：选中某个历史版本 → 恢复

### 跨团队协作

如果你的合作方也在 Wonderverse 上：

- 通过 **跨团队共享** 把单个项目临时给对方
- 资产 / 工作流可"输出"为单独包，发给对方导入

## 24. 提示词技巧

提示词（Prompt）是你和 AI 之间的**共同语言**。一个好的 prompt，能把生成质量从 60 分推到 90 分。
本节给你三类节点的 prompt 速成公式 + 避坑清单。

### 通用原则

| 原则 | 说明 |
|---|---|
| **具体 > 抽象** | "黄昏的金色阳光" > "好看的光" |
| **量化 > 形容** | "35mm 镜头景深虚化" > "有电影感" |
| **少即是多** | 关键词 6-12 个最佳，过多反而稀释 |
| **结构化** | 按"主体 + 动作 + 环境 + 风格 + 镜头"写 |
| **善用否定** | 用 negative prompt 排除不想要的元素 |

### 生图 · prompt 公式

```
[主体] + [动作 / 状态] + [环境 / 时间] + [视觉风格] + [镜头 / 光线]
```

**示例**：

| 维度 | 示例 |
|---|---|
| 主体 | 一位身着和服的少女 |
| 动作 | 轻倚樱花树 |
| 环境 | 京都春日清晨 |
| 风格 | 浮世绘风格 |
| 镜头 | 长焦虚化背景，柔光 |

合并：
> *"一位身着和服的少女，轻倚樱花树，京都春日清晨，浮世绘风格，长焦虚化背景，柔光"*

#### 风格关键词速查
- **写实摄影**：`写实摄影、35mm、对焦清晰、自然光`
- **日漫**：`日漫风格、赛璐璐着色、鲜艳色彩、夸张眼神`
- **3D 写实**：`3D 渲染、Octane、柔和阴影、次表面散射`
- **赛博朋克**：`赛博朋克、霓虹灯、雨夜、倒影、未来感`
- **水墨**：`水墨画、中国传统、极简、单色调`

### 生视频 · prompt 公式

视频比图像多一个核心要素：**运动**。

```
[主体 + 状态] + [运动方式] + [镜头运动] + [环境氛围] + [风格]
```

**示例**：

| 维度 | 示例 |
|---|---|
| 主体 | 一位老人坐在便利店 |
| 状态 | 阅读报纸，神情专注 |
| 主体运动 | 缓慢翻页，偶尔抬头 |
| 镜头运动 | 镜头从远景缓缓推进到中景 |
| 环境 | 雨夜，玻璃窗反光，霓虹倒影 |
| 风格 | 日式电影感，暖色调 |

#### 视频 prompt 避坑

| ❌ 不推荐 | ✅ 推荐 |
|---|---|
| 一个微笑的少女 | 少女缓缓转头，露出微笑 |
| 风景很美 | 镜头横摇，掠过雪山与湖泊 |
| 老人在看书 | 老人翻页阅读，背景偶有车声 |

### 配音 · prompt 技巧

文本本身就是 prompt，但有几个细节让效果飞跃：

#### 1. 标注情绪
```
[喜悦] "原来你也在这里！"
[悲伤、缓慢] "我以为，再也见不到你了……"
```

#### 2. 标注停顿
```
"那一年——（停顿1秒）—— 我还很年轻。"
```

#### 3. 多角色对白
```
A：你为什么回来？
B：(沉默) ……因为我答应过她。
```

### 漫剧分镜 · 进阶 prompt

漫剧分镜里，prompt 要兼顾**镜头语言**：

| 镜头类型 | Prompt 关键词 |
|---|---|
| 大全景 | `大全景、广阔风景` |
| 中景 | `中景、半身、对话场景` |
| 特写 | `特写、情绪细节、浅景深` |
| 主观镜头 | `主观镜头、第一人称视角` |
| 跟随镜头 | `跟随镜头、追拍主体` |
| 推拉 | `推近 / 拉远、戏剧性揭示` |

### 中 → 日

**语言风格**：
- 主语经常省略（一人称很少出现）
- 句尾敬语（です／ます）适配商业场景
- 漫剧偏好"だ"原型 + 拟声词丰富

**配音建议**：
- 漫剧女主 → 选「明朗 / 元气」音色
- 商业广告 → 选「知性 / 沉稳」音色
- 避免"翻译腔"——让翻译稿先过日方人审

**字幕样式**：
- 字体优先：`Yu Mincho`、`Hiragino Sans`、`Source Han Sans JP`
- 一行不超过 16 个全角字符
- 避免横排英文混排

### 中 → 英

**语言风格**：
- 主动语态优先
- 短句 + 节奏感（适合快剪短视频）
- 文化梗需要**等价翻译**而非直译

**配音建议**：
- 美式音色：年轻 / 清晰 / 有起伏
- 英式音色：沉稳 / 高级 / 适合奢侈品
- 中性国际口音：商业广告通用

**字幕样式**：
- 字体：`Inter`、`Helvetica`、`Roboto`
- 一行不超过 42 字符（含空格）
- 不要全大写（除了片名 / CTA）

### 中 → 韩

**语言风格**：
- 敬语层级严密（漫剧角色间称呼很重要）
- 短剧偏爱直接情感表达

**配音建议**：
- 漫剧 → 韩式甜美 / 干练音色
- 商业 → 标准首尔音

**字幕样式**：
- 字体：`Noto Sans KR`、`Apple SD Gothic`
- 行宽控制与日文相近

- ✅ 主图 1:1 + 场景图 3:4 是黄金组合
- ✅ 跨境电商至少出 3 个地域版本（亚 / 欧 / 美模特）

### 营销视频风格

| 风格 | 适合 | 标志特征 |
|---|---|---|
| 🔥 **燃向** | 游戏 / 体育 / 新能源车 | 快剪 + 重低音 BGM + 强对比色 |
| 💞 **治愈** | 美妆 / 母婴 / 茶饮 | 慢镜 + 钢琴 / 民谣 + 暖色 |
| 🎉 **幽默** | 食品 / 日常用品 / 校园品牌 | 反转 + 节奏感 + 鲜艳配色 |
| 💎 **高奢** | 奢侈品 / 珠宝 / 高端酒店 | 慢镜 + 极简构图 + 高级灰 |
| 🏃 **快节奏** | 短视频带货 / 闪购 | 0.5 秒一切 + 强字幕 + 紧凑配乐 |

#### 营销视频 · 黄金 3 秒法则
平台算法在前 3 秒决定推或不推。**第一帧必须是钩子**：

- ✅ 极致对比的画面（如冰山 vs 火焰）
- ✅ 反常识的一句话（"这件衣服我不让你买"）
- ✅ 强烈情绪（笑 / 哭 / 惊讶 / 愤怒）
- ❌ 慢慢出 Logo（除非奢侈品）

3. 设置名称 / 封面 / 关键词
4. 工作流绑定此风格 → 全团队产出统一调性

A：新账号注册赠送 1000 积分〔参数待核实〕，可用于约 30 张图 / 5 段视频。

### 生成质量

**Q：生成结果不满意怎么办？**
A：依次尝试：
1. 改 prompt
2. 调参数（提高分辨率 / 换风格预设）
3. 加参考图
4. 抽卡（2× / 4×）
5. 用 [超分增强](#9-超分增强) 后处理

**Q：人物角色一直变怎么办？**
A：把角色保存为 [资产卡](#22-个人资产)，在 prompt 里用 `@角色:名字` 锁定。

**Q：视频运动太僵硬 / 太剧烈？**
A：调整 [生视频](#8-生视频) 的「运动强度」参数。

### 时长与等待

**Q：为什么排队这么久？**
A：旺时（晚 8 点 - 凌晨）排队较长，建议设置**夜间批跑**让 Agent 自动排程。

**Q：批量任务能后台运行吗？**
A：可以。提交后关闭页面也会继续执行，完成后邮件通知。

**Q：能否一键删除所有数据？**

## 28. 计费与发票

Wonderverse 采用**订阅 + 用量**双模式，兼顾轻量用户与重度团队。
本节说明定价结构、积分规则、发票申请流程。

### 套餐结构

| 套餐 | 适合 | 月费〔示例〕 | 月度积分 | 团队人数 |
|---|---|---|---|---|
| **Free** | 个人尝鲜 | ¥0 | 1,000 | 1 |
| **Pro** | 独立创作者 | ¥199 | 12,000 | 1 |
| **Team** | 中小团队 | ¥999 / 团队 | 80,000 | 5 起，按人加费 |
| **Enterprise** | 大型企业 | 联系销售 | 自定义 | 无限 |

> ⚠️ 上述价格为示例〔真实价格待核实〕，请以官网为准。

### 积分计费规则

| 节点 | 积分消耗〔示例〕 |
|---|---|
| 生图 · 高清 | 5 / 张 |
| 生图 · 超高清 | 15 / 张 |
| 生视频 · 720p · 5s | 30 / 段 |
| 生视频 · 1080p · 30s | 200 / 段 |
| 超分增强 · 图 | 3 / 张 |
| 超分增强 · 视频 | 1 / 秒 |
| AI 译制配音 | 1 / 秒 |
| 字幕生成 / 译制 | 0.5 / 秒 |
| 智能擦除 · 视频 | 2 / 秒 |
| 横转竖 | 0.5 / 秒 |

### 积分查询与监控

- 顶栏显示当前剩余积分
- **资产中心 → 计费** 查看用量曲线 / 单项目消耗
- 团队管理员可设置成员配额（防止单人挥霍）

### 自动续费 / 包月规则

- 首次开通后默认按月自动续费
- 可随时在 **设置 → 订阅** 关闭自动续费
- 包月套餐内未用完的积分**不结转**到下月〔策略待核实〕

### 发票申请

1. 进入 **资产中心 → 计费 → 申请发票**
2. 选择开票月份 / 金额
3. 选择发票类型：
   - 增值税普通发票（电子）
   - 增值税专用发票（纸质，需邮寄）
4. 填写抬头、税号、邮箱 / 邮寄地址
5. 提交后 3-5 个工作日内出票

### 退款政策

- 月费套餐：开通 7 天内未实际使用积分，可全额退款〔策略待核实〕
- 年费套餐：按月折算
- 用量超额的积分包：售出不退

### 企业财务对接

企业版支持：

- 月结对账单
- API 用量明细导出（CSV / Excel）
- 多账号合并开票
- 跨主体计费〔企业版定制〕
